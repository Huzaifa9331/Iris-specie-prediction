Iris Species Prediction App

Developed by Huzaifa Masood

Project Objective
To build a Machine Learning model that can classify iris flowers into:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

and deploy it as a web application.
Technologies Used
- Python
- Machine Learning
- TensorFlow / Keras
- Streamlit
- Numpy & Scikit-learn
